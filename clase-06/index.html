Charts.js

Open source HTML5 Charts for your website

Para el gráfico que sigue, usamos datos de un JSON donde se describen a 10 personajes de StarWars. Ahora estamos probando Chart.js v3, una biblioteca de JavaScript distinta de p5.js, por lo que ya NO podemos tomar tales datos con un loadJSON(). Desde hoy usaremos la API Fetch de JavaScript.

Para el gráfico que sigue usamos un GeoJSON donde se describen todos los movimientos telúricos sobre 4.5M registrados por la USGS en el último mes. En este caso también tomamos los datos con la API Fetch de JavaScript. Se programa una consulta a cada registro de movimiento telúrico para contar los que incluyan "Chile" o "Japan" en su denominación (que refiere a su epicentro)

Para el gráfico que sigue usamos un CSV con las cifras nacionales y oficiales relacionadas con el Covid-19. En este caso también tomamos los datos con la API Fetch de JavaScript.

Para el gráfico que sigue, que es el último, aprovechamos otro CSV, al que llegamos gracias a uno de los videos de Daniel Shiffman: Zonal annual means, 1880-present, updated through most recent complete year.
